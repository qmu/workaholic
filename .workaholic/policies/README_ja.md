# ポリシー

7つのドメインにわたるリポジトリの実践を記述するポリシードキュメントです。ポリシードキュメントは scanner subagent によって生成されます。

- [Test](test_ja.md) - 検証と妥当性確認の戦略、テストレベル、カバレッジターゲット
- [Security](security_ja.md) - 認証、認可、シークレット管理、セキュリティ実践
- [Quality](quality_ja.md) - コード品質標準、リンティング、フォーマット、レビュープロセス
- [Accessibility](accessibility_ja.md) - 国際化、ローカライゼーション、コンテンツアクセシビリティ
- [Observability](observability_ja.md) - ログ、監視、メトリクス、トレーシング実践
- [Delivery](delivery_ja.md) - CI/CD パイプライン、リリースプロセス、デプロイメント実践
- [Recovery](recovery_ja.md) - バックアップ、災害復旧、データ復元実践

技術仕様については [../specs/](../specs/README_ja.md) を参照。
