# Absolute path enforcement, schema restructuring, and archive reliability

## Summary

Resolved three categories of infrastructure debt: runtime failures caused by relative skill script paths, a flat heading hierarchy in agent schemas, and unstaged file deletions caused by agents bypassing the archive script. Across 4 tickets and 10 commits, the work enforced absolute paths, restructured the Role/Goal/Responsibility hierarchy, and hardened the drive command's archive step.

## Key Changes

- Established a project-wide rule in CLAUDE.md mandating absolute paths for all skill shell script references
- Replaced relative paths with absolute paths across 39 files
- Restructured define-lead and define-manager schemas to nest Goal and Responsibility under Role
- Fixed unstaged ticket deletions by mandating explicit archive.sh invocation in the drive command
