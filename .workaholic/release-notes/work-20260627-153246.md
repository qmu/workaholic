# Finish the POSIX-sh re-enforcement etc

## Summary

This release hardens three internal contracts: it completes the POSIX-sh conversion of every bundled script and locks it behind a self-enforcing gate, reconnects the commit message body to `/report` so structured commit content reaches stories and release notes, and makes a default design-first `/trip` flow straight through to the build with no developer pause.

## Key Changes

- Converted the last bash scripts (including the `validate-ticket.sh` hook) to `#!/bin/sh -eu`, so the plugin runs on Alpine containers without bash
- Added a self-enforcing POSIX gate: a read-only lint auditor plus a dash/sh test runner with a regression-lock test
- Restructured the commit body around `Why`/`Changes`/`Concerns`/`Insights`/`Verify` and reconnected `collect-commits.sh`, which had been silently dropping the body
- Made a default design-first `/trip` proceed from its committed design into the build with no green-light pause

## Changes
