# Add missions and Codex hook compatibility

## Summary

Adds missions as a first-class Workaholic artifact with computed progress and workflow roll-ups. Also fixes deferred-concern publishing and the Codex hook-config parse failure.

- Added `/mission`, the `mission` skill, OKF indexing, and generated cross-agent workflow output.
- Added machine-readable mission/ticket relations across tickets, stories, and deferred concerns.

- Fixed deferred-concern extraction so post-merge commits are pushed to `origin/main`.
